# Bistros
Bistros projesi Onion Architecture yapısıyla, CQRS pattern ve MediatR kütüphanesini kullanılarak geliştirilmiş bir Restoran web tasarımıdır.
# Technologies
- MVC
- .Net Core Web API
- Entity Framework
- MediatR
- CQRS
- Object-Oriented Programming
